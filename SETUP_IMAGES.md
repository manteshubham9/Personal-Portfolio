# Adding Your Profile Image

## Quick Setup

To add your profile photo to the portfolio, follow these steps:

### 1. **Add Profile Image to the Images Folder**
   - Save your profile photo in the `images/` folder
   - Name it: `profile.jpg` (or `profile.png`)
   - Recommended size: 400x400 pixels or larger
   - File size: Keep under 200KB for better performance

### 2. **Supported Formats**
   - `.jpg` / `.jpeg` (recommended)
   - `.png`
   - `.webp`

### 3. **Image Placement**
The image will appear in two places:
- **Hero Section**: Full-screen background with overlay + centered circular image
- **About Section**: Smaller circular image on the left

### 4. **No Additional Coding Needed**
The portfolio is already configured to use `images/profile.jpg`. Just add the file and refresh your browser!

### 5. **Image Replacement**
To change the image, simply replace the `profile.jpg` file in the `images/` folder.

---

## Tips for Best Results

- **Aspect Ratio**: Use square images (1:1 ratio)
- **Background**: Plain or slightly blurred background works best
- **Quality**: Ensure good lighting and clear visibility
- **Format**: JPG is recommended for optimized file size
- **Dimensions**: 800x800 pixels minimum for high quality

---

## If Images Don't Load

1. Make sure the file is named exactly: `profile.jpg`
2. Check that it's in the `images/` folder (case-sensitive)
3. Hard refresh your browser: `Ctrl+Shift+R` (Windows) or `Cmd+Shift+R` (Mac)
4. Check browser console for any error messages
